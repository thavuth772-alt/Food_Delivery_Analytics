# Food Delivery Sales & Customer Insights

## Project Overview
This project analyzes food delivery sales data to identify:
- Peak order timings
- Popular food items
- Revenue trends
- Customer ordering patterns

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Revenue Analysis
- Popular Dish Identification
- Location-wise Order Analysis
- Peak Hour Detection
- Data Visualization Dashboard

## How to Run

```bash
python data/food_delivery_analysis.py
```
